HyperText : Anyone who has used the internet has probably used a Hyperlink They are used to help the users navigate the web.

Markup Breakdown

Skeleton

Every webpage you create should have these elements. These elements are the foundation that you will build on.

  • <!DOCTYPE html> : Declaration must be the very first thing in your HTML document, before the <html> tag

  • <html> : Defines the root of an HTML document

  • <head> : Defines information about the document

  • <body> : Represents the content of an HTML document 

     <!DOCTYPE html>
      <html>
        <head>

             information about your website,css files and script files usually go here

        </head>
        <body>

            HTML Markup will go inside the body tag.

        </body>
    </html>

Containers

The content on your page needs to be organized. The most common way to organize your content is with container elements. Here is a list of some of the most used elements to layout your webpages.

  • <main> : Specifies the main content of a document

  • <div> : Defines a division or a section

  • <section> : defines sections in a document, such as chapters, headers, footers, or any other sections of the document

  • <header> : Defines a header for a document or a section

  • <nav> : Defines a container for navigation links

  • <footer> : Defines a footer for a document or a section

  • <article> : Defines an independent self-contained article

  • <aside> : Defines content aside from the content (like a sidebar)

  • <details> : Defines additional details

  • <summary> : Defines a heading for the <details> element
